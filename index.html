MAMBEE - A Fábrica Escola de Software - IFPI

Um projeto de extensão realizado no Instituto Federal do Piauí Campus Picos (IFPI - Picos) que teve início em novembro de 2017. Funciona como um laboratório de prática de engenharia de software. Acreditamos que para inovação acontecer temos que os seguintes fatores:

  • Ambiente propício;
  • Pessoas criativas e estimuladas;
  • Um processo sistemático e contínuo de criação.

Novos integrantes como eu, somos inseridos em projetos reais que simulam o dia a dia de trabalho em uma empresa de tecnologia, estamos atualmente no início da criação de novo sistema para o refeitório do IFPI - Picos, colocando em pauta todos os critérios necessários para a criação desse novo sistema, com base no cliente (Nutricionista encarregada do Sistema). Estamos utilizando o Next.JS, framework do React, estamos aprendendo a utilizar do zero até chegarmos ao nosso real objetivo, com foco no trabalho em equipe e dedicação, o início desse projeto está nos colocando reais significado do que é buscar sempre mais conhecimentos.

React.JS

O React (também denominado React.js ou ReactJS) é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. Criado em 2011 pelo Facebook, com a criação de views declarativas e baseando-se em componentes, possuía o intuito de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social e melhorar a manutenção de código.

Next.JS

Next.js é uma estrutura da web de desenvolvimento front-end React de código aberto criada por Vercel que permite funcionalidades como renderização do lado do servidor e geração de sites estáticos para aplicativos da web baseados em React.

Cascading Style Sheets - CSS

Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags. Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos.
